Coop is a Neovim plugin that provides an asynchronous **op**eration framework
based on native Lua **co**routines.

## ⚡️ Requirements

- Neovim 0.9+

## 📦 Installation

Install the plugin with your preferred package manager, such as [Lazy]:

```lua
{
  "gregorias/coop.nvim",
}
```
